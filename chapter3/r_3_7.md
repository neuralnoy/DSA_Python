## Equivalence of the Two Statements

Let T_A(x) be the running time of algorithm A on input x, and let

T(n) = max over all inputs x with |x| = n of T_A(x)

denote the worst-case running time for inputs of size n.

---

### (a) implies (b)

**Statement (a):** The running time of algorithm A is always O(f(n)).

This means there exist constants c > 0 and n0 such that

T_A(x) <= c * f(n)

for every input x with size n >= n0.

Since this inequality holds for every input of size n, it also holds for the maximum running time among them:

T(n) <= c * f(n)

Therefore, the worst-case running time is O(f(n)), which proves statement (b).

---

### (b) implies (a)

**Statement (b):** The worst-case running time of algorithm A is O(f(n)).

So there exist constants c > 0 and n0 such that

T(n) <= c * f(n)

for all n >= n0.

By definition of worst-case running time, for any specific input x of size n:

T_A(x) <= T(n)

Combining the inequalities:

T_A(x) <= T(n) <= c * f(n)

This shows that the running time on every input is bounded by O(f(n)), which proves statement (a).

---

## Conclusion

The two statements are equivalent:

"Algorithm A runs in O(f(n)) for all inputs"
if and only if
"The worst-case running time of algorithm A is O(f(n))".
